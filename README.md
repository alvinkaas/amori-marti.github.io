# amori-marti.github.io
# Дизайн-концепция сайта питомника "Амори Марти"

## Общая концепция
Сайт выполнен в стиле элегантной редакции журнала премиум-класса, сочетающей классическую эстетику с современными веб-технологиями. Дизайн подчеркивает престиж питомника и высокое качество разведения цвергшнауцеров.

## Цветовая палитра
- **Основной цвет**: Темно-синий (#1a365d) - надежность и престиж
- **Вторичный цвет**: Теплый серый (#4a5568) - элегантность
- **Акцентный цвет**:  - премиум качество
- **Фоновый цвет**: Кремовый (#faf5f0) - уют и тепло
- **Текст**: Черный (#2d3748) для основного текста, белый (#ffffff) для текста на темном фоне

## Типографика
- **Заголовки**: Playfair Display (serif) - классическая элегантность
- **Основной текст**: Inter (sans-serif) - современная читаемость
- **Акценты**: Montserrat (sans-serif) - стабильность и надежность

## Визуальный стиль
- Минималистичный дизайн с акцентом на качественные изображения
- Просторные макеты с изысканной типографикой
- Плавные анимации и эффекты для создания премиального опыта
- Классическая компоновка в духе элитных журналов

## Эффекты и анимации
- Плавное появление контента при прокрутке
- Эффект параллакса для героической секции
- Анимированные карточки преимуществ
- Интерактивная галерея изображений
- Плавные переходы между секциями
